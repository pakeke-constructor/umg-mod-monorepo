start

  • start(samplecount: number, samplerate?: number, bitdepth?: number, channels?: number): boolean
    • Defined in love.audio/types/RecordingDevice.d.ts:86

    Begins recording audio using this device.

    link

    RecordingDevice:start

    since

    11.0

    Parameters

    • samplecount: number

      The maximum number of samples to store in an internal ring buffer when recording. RecordingDevice:getData clears the internal buffer when called.

    • Optional samplerate: number

      The number of samples per second to store when recording. (Default 8000)

    • Optional bitdepth: number

      The number of bits per sample. (Default 16)

    • Optional channels: number

      Whether to record in mono or stereo. Most microphones don't support more than 1 channel. (Default 1)

    Returns boolean

    success, true if the device successfully began recording using the specified parameters, false if not.
